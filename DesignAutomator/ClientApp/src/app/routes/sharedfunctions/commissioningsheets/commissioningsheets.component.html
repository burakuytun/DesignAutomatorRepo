Use Source Addressing

Select when migrating an existing system "hardware sheet" into different commissioning sheets. Compatible commissioning sheets* will be populated with the devices in the same order as existing writing.

Use Source Naming

Select when migrating an existing system "hardware sheet" into different commissioning hseets. Compatible commissioning sheets* will retain the existing system naming convention and not adopt the "DNA" naming convention.

System Type

Select the system you wish to produce commissioning sheets for. The system type relates to the manufacturer/name of the access control system.

Panel Type

Select the type of panel you wish to produce commissioning sheets for. The panel type will include a list of supported "panels", also known as "controllers"

Reader Connection Type

Select the type of the reader connection you wish to produce commissioning sheets for. The reader connection type indicates the connection type used when connecting readers, for example "directly connected to the panel" or "connected to a remote board", etc.

Number of Boards Per Controller

Select the number of "boards" you wish to specify for each single access control panel. This allows you to reduce the number of physical circuit boards that will be used on a per panel basis. The default is set to the maximum "boards" that the panel can control.

Number of Reader Ports Per Board

Select the number of "ports" you wish to specify for use for each "board". This allows you to reduce the number of connections that will be utilised on each "board". The connections are divided into "reader" ports. The default is set to the maximum reader "ports" that the "board" physically has.
